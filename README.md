# ESLint config Namics


## Installation

```bash
$ npm install --save-dev eslint @namics/eslint-config
```


## Usage

- `namics` - ES6 + browser
- `namics/configurations/es5-browser` - ES5 + browser
- `namics/configurations/es6-react` - ES6 + react
